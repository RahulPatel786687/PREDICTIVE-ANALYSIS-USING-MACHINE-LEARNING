*DESCRIPTION*: 
This project is a Machine Learning based House Price Prediction system developed using Python and Scikit-learn. The main objective of this project is to predict housing prices based on different features from the California Housing Dataset. It demonstrates the complete workflow of a Regression Machine Learning project, including data preprocessing, exploratory data analysis (EDA), model training, evaluation, and visualization.

The project begins by importing essential Python libraries such as Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn. The California Housing Dataset is loaded and converted into a Pandas DataFrame for easier analysis and manipulation. Basic dataset information such as shape, statistical summary, and missing values are explored to understand the dataset better.

Exploratory Data Analysis (EDA) is performed using visualizations like correlation heatmaps and price distribution graphs. These visualizations help identify relationships between features and understand the overall behavior of housing prices.

The dataset is then divided into independent variables (features) and the target variable (house price). After preprocessing, the data is split into training and testing sets using the train-test split method.

# Two regression models are implemented in this project:

Linear Regression
Random Forest Regressor

The Linear Regression model is trained first to create a baseline prediction model. Predictions are generated and evaluated using metrics such as Mean Absolute Error (MAE), Mean Squared Error (MSE), Root Mean Squared Error (RMSE), and R² Score.

To improve prediction accuracy, a Random Forest Regressor model is also implemented. This model provides better performance and more accurate predictions compared to Linear Regression. Feature importance analysis is also included to identify the most influential factors affecting house prices.

The project includes various graphs and plots for better data understanding and result visualization. It provides hands-on experience with Regression algorithms, data analysis, predictive modeling, and Machine Learning workflows.
